Project Details Vision Transformer

Vision Transformer using CIFAR-10 Dataset

Overview: This project involves creating a Vision Transformer (ViT) model capable of recognizing images from the CIFAR-10 dataset. The model leverages the power of transformers to classify images into one of the ten categories in CIFAR-10.

Technologies Used: Python, NumPy, Pandas, TensorFlow, Keras, Jupyter Notebook

Features:

  • Data preprocessing and augmentation
  • Vision Transformer model implementation
  • Model training and evaluation
  • Visualization of results

Challenges: One of the main challenges was ensuring the model converges properly while optimizing for accuracy and minimizing training time. Data augmentation and proper hyperparameter tuning were key to achieving good performance.
